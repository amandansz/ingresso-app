#  e-Ticket – Sistema de Compra de Ingressos

Este é um projeto desenvolvido durante o curso de **Lógica de Programação com JavaScript** da [Alura](https://www.alura.com.br/), com foco no uso de estruturas condicionais, funções, manipulação de DOM e boas práticas em JS.

A interface simula um sistema de venda de ingressos com controle de quantidade e tipos de assentos. Também conta com um visual moderno usando **HTML5**, **CSS3** e **JavaScript puro**.

---

##  Conceitos Trabalhados

- Manipulação do DOM com JavaScript
- Funções, condicionais e eventos
- Validação de entradas
- Atualização dinâmica de conteúdo (quantidade de ingressos)
- Design responsivo com HTML e CSS
- Organização modular de arquivos

---

##  Funcionalidades

- Seleção do tipo de ingresso (pista, cadeira superior ou inferior)
- Definição da quantidade de ingressos desejada
- Validação de disponibilidade antes da compra
- Atualização automática da quantidade restante
- Interface estilizada com elementos gráficos e fontes personalizadas
